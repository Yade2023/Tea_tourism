# 🍵 Tea Tourism - 台灣茶旅網站

一個展示台灣茶文化的現代化網站，提供茶葉知識、茶旅專區、聯絡服務等功能。

## 📋 項目概述

茶旅網站是一個以 Vue.js 3 + Vite 構建的現代化單頁應用，致力於推廣台灣茶文化，為茶愛好者提供全面的茶葉資訊和旅遊服務。

## ✨ 主要功能

### 🏠 首頁 (HomeTea_tourism)
- **精美首頁展示**：茶文化主題設計
- **導航功能**：快速導向各功能區塊

### 🧭 智能導航系統
- **響應式導航欄**：支援滑鼠懸停自動顯示/隱藏
- **SVG波浪設計**：獨特的波浪型底部效果 (#d2f3cd 茶綠色)
- **智能回到頂部**：垂直置中的 TOP 按鈕，右側固定位置

### 📚 台灣茶知識庫 (knowledge)
- **6種台灣代表茶類**：
  - 紅茶 (全發酵茶)
  - 培香烏龍 (半發酵重焙火)
  - 東方美人茶 (重發酵輕焙火)
  - 烏龍茶 (半發酵茶)
  - 包種茶 (輕發酵茶)
  - 綠茶 (不發酵茶)
- **互動式展示**：點擊茶葉卡片展開詳細資訊
- **完整茶葉資訊**：特色、湯色、代表茶品、產地、功效
- **專業沖泡指南**：標準茶葉沖泡流程圖解
- **茶葉特色輪播**：6種茶湯展示與特色說明
- **台灣茶發展史**：6個歷史階段完整介紹
- **響應式按鈕**：平板/手機版浮動式左右切換按鈕

### 🗺️ 茶旅專區 (teaTourismArea)
- **茶園景點推薦**
- **茶文化體驗活動**
- **旅遊路線規劃**

### 📞 關於我們 (address)
- **智能FAQ系統**：手風琴式展開常見問答
- **API動態整合**：
  - 支援 `http://localhost:5028/api/Address` API呼叫
  - API成功/失敗狀態顯示
  - Fallback機制：API失敗時使用本地JSON資料
- **聯絡表單**：姓名、信箱、訊息完整表單
- **即時狀態反饋**：根據API狀態動態顯示標題

### 👤 登入系統 (login)
- **用戶登入介面**
- **身份驗證功能**

## 🛠️ 技術架構

### 核心技術棧
- **Vue.js 3.5.22**：使用 Composition API + `<script setup>`
- **Vue Router 4.5.1**：SPA路由管理
- **Vite 7.1.7**：現代化構建工具
- **Bootstrap 5.3.8**：UI框架支援
- **CSS3**：Grid/Flexbox響應式布局
- **ES6+ JavaScript**：現代化開發語法

### 響應式斷點設計
- **大螢幕**：≥1200px (padding: 0 3rem)
- **筆電**：992px-1199px (padding: 0 1.5rem)
- **平板**：768px-991px (垂直布局)
- **小平板**：481px-767px (標題居中)
- **手機**：≤480px (優化觸控體驗)

## 🚀 快速開始

### 環境要求
- **Node.js** 16.0 或更高版本
- **npm** 或 **yarn** 套件管理器

### 安裝與運行

1. **克隆專案**
```bash
git clone https://github.com/Yade2023/Tea_tourism.git
cd Tea_tourism
```

2. **安裝相依套件**
```bash
npm install
```

3. **啟動開發伺服器**
```bash
npm run dev
```
- 開發伺服器預設運行在 `http://localhost:5173`

4. **建置生產版本**
```bash
npm run build
```

5. **預覽生產版本**
```bash
npm run preview
```

## 📱 特色功能詳解

### 🎨 導航欄設計特色
- **滾動隱藏**：向下滾動時自動隱藏，提升閱讀體驗
- **智能顯示**：滑鼠移至頂部60px區域自動顯示
- **SVG波浪效果**：底部波浪造型，z-index層級管理
- **完全響應式**：所有裝置尺寸完美適配

### 🍵 茶葉知識系統
- **視覺化展示**：每種茶葉精美圖片與詳細介紹
- **專業內容**：包含發酵程度、香氣特色、沖泡方法
- **互動體驗**：
  - 點擊展開模式：1/6圖片 + 5/6詳細資訊
  - 響應式佈局：手機版改為垂直排列
- **輪播功能**：茶湯特色與歷史文化雙重輪播

### 🔌 API整合機制
- **動態載入**：支援外部API數據更新
- **智能合併**：`mergeDefault.js` 實現API與本地數據合併
- **錯誤處理**：API失敗時seamless fallback到本地JSON
- **狀態管理**：即時顯示API呼叫成功/失敗狀態
